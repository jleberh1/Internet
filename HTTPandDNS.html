This page conatins information on Hypertext Transfer Protocol and the Domain Name System

What is Hypertext Transfer Protocol?

Hypertext Transfer Protocol is a protocol used fir sending data across the internet. The system allows browsers and data from a web server to sent to the requested browser. The Hypertext Transfer Protocol can also do client-server communication which is the machine that hosts the website or content such ad Google, or Safari, and Firefox.

What is Domain Name System?

The Domain name system is a human-readible system that takes domain names like www.example.com into a machine readible addresses which are numerical. The Domain Name System can find where to find domians like .com, .org and many more. When looking up a website the Domain Name System returns the IP address in a machine readable way.
